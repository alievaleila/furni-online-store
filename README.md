# Furni Online Store 🛋️

FURNI is a high-performance e-commerce platform built with **Spring Boot MVC**. It features a secure **Spring Security** layer, clean **DTO architecture**, and an optimized **Shopping Cart** management system.

## 🚀 Key Features
* **Product Catalog:** Browse items with real-time availability.
* **Shopping Cart:** Advanced cart logic (automatic quantity updates).
* **Security:** Role-based access control and secure authentication.
* **DTO Pattern:** Decoupled data layers for better maintainability.

## 🛠️ Tech Stack
* **Backend:** Java 17, Spring Boot 3.x
* **Database:** PostgreSQL (Production)
* **Security:** Spring Security
* **Template Engine:** Thymeleaf

## ⚙️ Setup & Installation
1. Clone the repository: `git clone https://github.com/alievaleila/furni-online-store.git`
2. Update database credentials in `application.properties`.
3. Run the application: `mvn spring-boot:run`

## Environment variables
Ensure the following variables are set in your environment or application.properties:

SPRING_DATASOURCE_URL: jdbc:postgresql://localhost:5432/furni

SPRING_DATASOURCE_USERNAME: postgres

SPRING_DATASOURCE_PASSWORD: 12345

SERVER_PORT: Application port (default: 8080).
